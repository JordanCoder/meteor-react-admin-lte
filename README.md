### Installation

1. Clone this repo to `<yourapp>`

  `git clone https://github.com/jeescu/meteor-react-adminLTE-starter.git <yourapp>`

2. Remove `.git`

  `cd <yourapp> && rm -rf .git`

3. Start coding!
